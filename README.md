# Prodder — moved to sshTools

**This project now lives in the [sshTools](https://github.com/reygnn/sshTools) monorepo.**

Prodder is the session prodder that captures and feeds running `screen` sessions — one of four small Android remotes for a single trusted SSH
build host. They were merged into one Gradle build so they can share their SSH,
crypto, persistence and onboarding code through a common set of `core-*`
modules, while still shipping as independent apps.

➡️ **Code, releases and issues:**
**[github.com/reygnn/sshTools](https://github.com/reygnn/sshTools)** — module `app-prodder`.

it is no longer maintained.
